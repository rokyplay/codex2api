一、隐私条款

  1. 为完成身份验证与访问控制，我们会收集并处理第三方 OAuth 账号的基本信息，包括但不限于：用户名、账号 ID、头像信息。
  2. 为保障服务安全与合规运行，我们会记录全部 API 请求的运行元数据，包括但不限于：请求时间、所用模型、Token 数量等。
  3. 我们不收集、不存储、不审核用户的对话内容（prompt 和 response）。我们仅收集行为元数据（请求时间、频率、Token 用量、IP 哈希、错误率等）用于自动化风险识别与滥用检测。行为数据保留 2 天后自动清理。
  4. 上述数据存储于我们控制的云服务器环境中，并采取合理的安全措施进行保护。
  5. 请求数据主要用于 AI 自动化检测与风控流程，不进行人工逐条审阅；但在出现安全事件、合规要求或系统故障排查时，可能进行必要范围内的审计访问。
  6. 我们不会将您的数据出售给任何第三方，也不会向第三方提供可识别个人身份的数据，法律法规另有要求的除外。
  7. 您一旦使用本服务，即视为已阅读、理解并同意本隐私条款的全部内容。
